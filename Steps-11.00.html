Set Up Staging Layer ETL Package 11


Detailed Instructions:

Step 1: Create a SISS package. 

A. Named it:  1-Load_Collision_Stg

B-On the Control Flow pane,  drag and drop one Execute SQL Task (named Truncate_Collusion_Stg) and a Sequence Container.

C-Drag and drop a Data Flow Tasks  and Execute SQL Task   into the Sequence Container and name them as shown below



Step 2: Create a Connection to the Seattle_Collisions _Staging Database.

A. Right click in the Connection Manager section.

B. Select New OLE DB Connection.

C. Select Seattle_Collisions _Staging.



Step 3: Configure both execute SQL task 

A-Double-click on the Execute SQL Task Truncate_Collision_Stg

Click on Connection to select the correct connection manager

Click on the 3 Dots in the SQL Statement box to write SQL Truncate statement  (Truncate Table Collison_Stg)

B-Double-click on the Execute SQL Task Update_Collision_Stg; Write the three Update statement


Step 4:Connect the both execute SQL task
A- Connect Execute SQL Task Truncate_Collision_Stg to the the sequence container as shown below
B- Connect Execute SQL Task  Update_Collision_Stg  to the data flow task : DFT-Load_Collision_Stg as shown below
